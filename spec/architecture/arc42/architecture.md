# Архитектурная документация arc42

Этот каталог содержит инженерно-подробное архитектурное описание Unica в
структуре arc42.

Документ разделен на части, чтобы AI-агентам и контрибьюторам было проще
ссылаться на конкретные архитектурные факты, поддерживать sections независимо и
не редактировать один большой файл.

## Как использовать

- Для быстрого понимания системы начинайте с разделов 1, 3, 4 и 5.
- Для анализа tool call lifecycle смотрите раздел 6.
- Для безопасных изменений MCP, adapters, cache и skill routing особенно важны
  разделы 8, 9, 10 и 11.
- Если архитектурное решение еще не оформлено как ADR, считайте source of truth
  код и соответствующий раздел arc42.
